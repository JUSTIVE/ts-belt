### add

Returns the sum of two numbers.


```ts
function add(b: number): (a: number) => number
function add(a: number, b: number): number
```

```ts
N.add(10, 20) // → 30
pipe(5, N.add(10)) // → 15
```


```ts
N.divide(20, 5) // → 4
pipe(40, N.divide(4)) // → 10
```

### divideWithModulo

Returns the quotient and remainder of a number division.


```ts
function divideWithModulo(b: number): (a: number) => [number, number]
function divideWithModulo(a: number, b: number): [number, number]
```

```ts
N.divideWithModulo(20, 5) // → [4, 0]
pipe(30, N.divideWithModulo(4)) // → [7.5, 2]
```
